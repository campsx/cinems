#Le CMS **CineMS**

##Installation

Pour commencer, il faut installer le CMS, il faut remplire le fichier config.ini dans le dossier core/config.ini

```
[data_base_section]
DB_USER = root
DB_PWD = root
DB_NAME = cinems
DB_HOST = localhost
DB_PORT = 8889
DB_TYPE = mysql
 
[mod]
ENV_TYPE = dev
```
